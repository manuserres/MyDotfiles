# 󰣇 My Dotfiles - Omarchy Arch Linux

Configuraciones personales para un entorno basado en Hyprland.

##  Carpetas Sincronizadas
- **bash**: Configuración del shell y alias.
- **hypr**: Window Manager (Hyprland), atajos, apariencia e inactividad.
- **waybar**: Barra de estado superior.
- **fastfetch**: Información del sistema al abrir la terminal.
- **ghostty**: Terminal principal.
- **nvim**: Editor de texto Neovim.
- **mako**: Notificaciones del sistema.
- **walker**: Lanzador de aplicaciones y menús.
- **swayosd**: Indicadores visuales de volumen/brillo.
- **btop / lazygit**: Herramientas de monitoreo y Git.
